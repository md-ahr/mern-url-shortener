## URL Shortener
### A simple mern stack project for using url shortener

#### Technology used

`Backend: express js, mongodb`
`Frontend: react js`

##### Run Rest API Server

Install yarn package globally

```sh
   npm install yarn -g
```

Go to backend directory

```sh
   cd backend
```
Install package dependencies

```sh
   yarn
```

Start the server

```sh
   yarn start
```

Build

```sh
   yarn build
```
